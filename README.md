# ai_js_starter

Opinionated JS/TS starter template optimized for AI-assisted development with Claude Code.

## What's Included

- **[CLAUDE.md](./CLAUDE.md)** — AI coding standards, persona system, and project scaffold
- **[DESIGN_PRINCIPLES.md](./DESIGN_PRINCIPLES.md)** — Mobile-first, accessible UI/UX rules (WCAG 2.1 AA+)
- **Claude Code settings** — Permissions, auto-format hook (Prettier on save), TypeScript LSP plugin
- **Worktree support** — Script and gitignore setup for parallel AI branches
- **Prettier** — Pre-configured with sensible defaults

## Quick Start

```bash
# Use as a template on GitHub, then:
git clone <your-repo-url>
cd <your-repo>
pnpm install
```

## Usage

This is a starting point, not a framework. Add your own tooling (TypeScript, React, Node, etc.) on top. The AI configuration works regardless of what you build.
